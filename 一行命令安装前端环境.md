# 前端环境一键安装 Skill

帮助用户快速搭建完整的前端开发环境，支持 macOS、Linux、Windows 多平台。

## 功能特性

- ✅ 自动检测操作系统
- ✅ 安装 Node.js（支持指定版本）
- ✅ 安装包管理器（npm/yarn/pnpm）
- ✅ 配置 npm 镜像源（淘宝镜像等）
- ✅ 安装常用全局工具
- ✅ 验证安装结果

## 使用流程

### 1. 检测当前环境

首先检测用户当前的操作系统和已安装的前端工具：

```bash
# 检测操作系统
uname -s  # macOS/Linux
# 或在 Windows 上
echo %OS%

# 检测已安装的工具
node --version 2>/dev/null || echo "Node.js 未安装"
npm --version 2>/dev/null || echo "npm 未安装"
yarn --version 2>/dev/null || echo "yarn 未安装"
pnpm --version 2>/dev/null || echo "pnpm 未安装"
```

### 2. 安装 Node.js

根据操作系统选择合适的安装方式：

#### macOS 推荐方式

**方式一：使用 Homebrew（推荐）**
```bash
# 安装 Homebrew（如果未安装）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node

# 或安装特定版本
brew install node@18
brew install node@20
```

**方式二：使用 nvm（Node Version Manager）**
```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新加载终端配置
source ~/.zshrc  # 或 source ~/.bashrc

# 安装 Node.js LTS 版本
nvm install --lts

# 或安装特定版本
nvm install 18
nvm install 20
nvm use 18
```

#### Linux 推荐方式

**方式一：使用 nvm**
```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新加载终端配置
source ~/.bashrc

# 安装 Node.js
nvm install --lts
```

**方式二：使用包管理器**
```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# CentOS/RHEL
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -
sudo yum install -y nodejs
```

#### Windows 推荐方式

**方式一：使用官方安装包**
- 访问 https://nodejs.org/
- 下载 LTS 版本安装包
- 运行安装程序

**方式二：使用 Chocolatey**
```powershell
# 安装 Chocolatey（如果未安装）
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# 安装 Node.js
choco install nodejs
```

**方式三：使用 fnm（Fast Node Manager）**
```powershell
# 安装 fnm
winget install Schniz.fnm

# 配置 fnm
fnm env --use-on-cd | Out-String | Invoke-Expression

# 安装 Node.js
fnm install --lts
```

### 3. 安装包管理器

Node.js 安装后会自带 npm，可以选择性安装 yarn 或 pnpm：

```bash
# 安装 yarn
npm install -g yarn

# 安装 pnpm
npm install -g pnpm
```

### 4. 配置 npm 镜像源

为提高国内下载速度，建议配置镜像源：

```bash
# 配置淘宝镜像
npm config set registry https://registry.npmmirror.com

# yarn 配置镜像
yarn config set registry https://registry.npmmirror.com

# pnpm 配置镜像
pnpm config set registry https://registry.npmmirror.com

# 验证配置
npm config get registry
```

**恢复官方源：**
```bash
npm config set registry https://registry.npmjs.org
```

### 5. 安装常用全局工具

根据用户需求安装常用的前端开发工具：

```bash
# Vue 开发
npm install -g @vue/cli
npm install -g vite

# React 开发
npm install -g create-react-app

# 通用工具
npm install -g typescript
npm install -g ts-node
npm install -g nodemon
npm install -g pm2

# 代码质量工具
npm install -g eslint
npm install -g prettier

# 构建工具
npm install -g webpack webpack-cli
npm install -g rollup
npm install -g parcel

# 其他常用工具
npm install -g http-server
npm install -g serve
npm install -g npx
```

### 6. 验证安装

验证所有工具是否正确安装：

```bash
echo "=== Node.js 版本 ==="
node --version

echo "=== npm 版本 ==="
npm --version

echo "=== 包管理器版本 ==="
yarn --version 2>/dev/null || echo "yarn 未安装"
pnpm --version 2>/dev/null || echo "pnpm 未安装"

echo "=== npm 镜像源 ==="
npm config get registry

echo "=== 全局工具版本 ==="
vue --version 2>/dev/null || echo "@vue/cli 未安装"
tsc --version 2>/dev/null || echo "TypeScript 未安装"
```

## 完整安装脚本

### macOS/Linux 自动化脚本

创建并执行以下脚本，一键完成所有安装：

```bash
#!/bin/bash

# 前端环境一键安装脚本（macOS/Linux）

echo "🚀 开始安装前端开发环境..."

# 检测操作系统
OS="$(uname -s)"
echo "检测到操作系统: $OS"

# 检测是否已安装 Node.js
if command -v node &> /dev/null; then
    echo "✅ Node.js 已安装: $(node --version)"
else
    echo "📦 正在安装 Node.js..."
    
    # macOS 使用 Homebrew
    if [[ "$OS" == "Darwin" ]]; then
        if command -v brew &> /dev/null; then
            brew install node
        else
            echo "❌ 请先安装 Homebrew: /bin/bash -c \"\$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)\""
            exit 1
        fi
    fi
    
    # Linux 使用 nvm
    if [[ "$OS" == "Linux" ]]; then
        if ! command -v nvm &> /dev/null; then
            curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
            export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
            [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
        fi
        nvm install --lts
    fi
fi

# 配置 npm 镜像源（淘宝）
echo "⚙️  配置 npm 镜像源..."
npm config set registry https://registry.npmmirror.com

# 安装 yarn 和 pnpm
echo "📦 安装 yarn 和 pnpm..."
npm install -g yarn
npm install -g pnpm

# 安装常用全局工具
echo "📦 安装常用全局工具..."
npm install -g @vue/cli
npm install -g vite
npm install -g create-react-app
npm install -g typescript
npm install -g eslint
npm install -g prettier
npm install -g nodemon
npm install -g pm2
npm install -g http-server

# 验证安装
echo ""
echo "✅ 安装完成！"
echo ""
echo "=== 环境信息 ==="
echo "Node.js: $(node --version)"
echo "npm: $(npm --version)"
echo "yarn: $(yarn --version)"
echo "pnpm: $(pnpm --version)"
echo "npm 镜像源: $(npm config get registry)"
echo ""
echo "=== 全局工具 ==="
echo "Vue CLI: $(vue --version 2>/dev/null || echo '未安装')"
echo "TypeScript: $(tsc --version 2>/dev/null || echo '未安装')"
```

### Windows PowerShell 脚本

```powershell
# 前端环境一键安装脚本（Windows）

Write-Host "🚀 开始安装前端开发环境..." -ForegroundColor Green

# 检测是否已安装 Node.js
if (Get-Command node -ErrorAction SilentlyContinue) {
    Write-Host "✅ Node.js 已安装: $(node --version)" -ForegroundColor Green
} else {
    Write-Host "📦 正在安装 Node.js..." -ForegroundColor Yellow
    
    # 使用 Chocolatey
    if (Get-Command choco -ErrorAction SilentlyContinue) {
        choco install nodejs -y
    }
    # 使用 winget
    elseif (Get-Command winget -ErrorAction SilentlyContinue) {
        winget install OpenJS.NodeJS.LTS
    }
    else {
        Write-Host "❌ 请先安装 Chocolatey 或 winget，或手动下载 Node.js 安装包" -ForegroundColor Red
        Write-Host "下载地址: https://nodejs.org/" -ForegroundColor Cyan
        exit 1
    }
}

# 配置 npm 镜像源（淘宝）
Write-Host "⚙️  配置 npm 镜像源..." -ForegroundColor Yellow
npm config set registry https://registry.npmmirror.com

# 安装 yarn 和 pnpm
Write-Host "📦 安装 yarn 和 pnpm..." -ForegroundColor Yellow
npm install -g yarn
npm install -g pnpm

# 安装常用全局工具
Write-Host "📦 安装常用全局工具..." -ForegroundColor Yellow
$tools = @(
    "@vue/cli",
    "vite",
    "create-react-app",
    "typescript",
    "eslint",
    "prettier",
    "nodemon",
    "pm2",
    "http-server"
)

foreach ($tool in $tools) {
    npm install -g $tool
}

# 验证安装
Write-Host ""
Write-Host "✅ 安装完成！" -ForegroundColor Green
Write-Host ""
Write-Host "=== 环境信息 ===" -ForegroundColor Cyan
Write-Host "Node.js: $(node --version)"
Write-Host "npm: $(npm --version)"
Write-Host "yarn: $(yarn --version)"
Write-Host "pnpm: $(pnpm --version)"
Write-Host "npm 镜像源: $(npm config get registry)"
```

用户可以指定要安装的 Node.js 版本：

```bash
# 使用 nvm 安装特定版本
nvm install 18.17.0
nvm use 18.17.0
nvm alias default 18.17.0

# 使用 fnm 安装特定版本
fnm install 20.10.0
fnm use 20.10.0
```

## 故障排除

### 常见问题

1. **权限问题**
   ```bash
   # macOS/Linux 修复 npm 权限
   sudo chown -R $(whoami) ~/.npm
   sudo chown -R $(whoami) /usr/local/lib/node_modules
   ```

2. **网络问题**
   ```bash
   # 切换镜像源
   npm config set registry https://registry.npmmirror.com
   
   # 或使用代理
   npm config set proxy http://proxy.example.com:8080
   npm config set https-proxy http://proxy.example.com:8080
   ```

3. **版本冲突**
   ```bash
   # 清理 npm 缓存
   npm cache clean --force
   
   # 使用 nvm 管理多版本
   nvm install 18
   nvm use 18
   ```
